# ZkC / Corset

This repository hosts a compiler for the ZkC language, and maintains
contains a legacy compiler for the (older) Corset language. ZkC is a
simple imperative language designed primarily for writing programs
whose executions can be proved (e.g. using a [zero-knowledge proof
system](https://en.wikipedia.org/wiki/Zero-knowledge_proof) or
similar). ZkC provides a simple syntax with fixed-width unsigned
integer types and explicit memory declarations.
